"Dime con quién andas..."

Tecnología: Inductive Graph Neural Networks

El fraude es contagioso. Los sistemas antiguos miran a la empresa aislada. Este algoritmo mira con quién se junta. Si una empresa nueva entra al banco y parece perfecta, pero sus proveedores son los mismos que usó una trama criminal hace 3 años, el sistema la marca antes de que pueda operar.

gnn_model.py
# PyTorch Geometric Implementation
import torch
from torch_geometric.nn import SAGEConv

class FraudGNN(torch.nn.Module):
  def __init__(self, in_channels, hidden_channels):
    super().__init__()
    # Agregamos vecinos a 2 saltos (Hop-2) para ver "amigos de amigos"
    self.conv1 = SAGEConv(in_channels, hidden_channels)
    self.conv2 = SAGEConv(hidden_channels, 64) # Embedding latente

Detector de "Maquillaje Contable"

Tecnología: Variational Autoencoders

Funciona como un experto en arte que detecta una falsificación perfecta. Los defraudadores saben cuadrar el Excel, pero no pueden replicar la complejidad matemática natural de un negocio real. Este modelo detecta la artificialidad estadística en balances que, a simple vista, parecen impecables.

anomaly_detector.py
def get_reconstruction_error(model, balance_sheet):
  with torch.no_grad():
    # Intentamos comprimir y descomprimir el balance
    encoded, mu, logvar = model.encoder(balance_sheet)
    reconstructed = model.decoder(encoded)

  # Si el error es alto, el balance es "artificial"
  loss = torch.nn.functional.mse_loss(reconstructed, balance_sheet)
  if loss > THRESHOLD: return "FRAUDE_MAQUILLAJE"

El Historiador (Fraude Durmiente)

Tecnología: Temporal Fusion Transformers

El fraude no es una foto fija, es una película. Muchos defraudadores se comportan bien durante años para ganar confianza y dar un "golpe final" (Bust-out). Este modelo analiza la historia completa de 5 años y detecta cambios sutiles de comportamiento hoy que predicen una fuga mañana.
